## Codebase Patterns
- PHPStan level 8: use `/** @var resource */` or helper methods to narrow types from nullable properties after guard methods
- Sender interface requires: `post(string $endpoint, string $apiToken, array $payload, FlareEntityType $type, bool $test, Closure $callback): void`
- DaemonSender at `src/Senders/DaemonSender.php` - implements TCP payload protocol with format `{total_length}:v1:{type}:{jsonPayload}`
- DaemonTimeoutException at `src/Senders/Exceptions/DaemonTimeoutException.php` - thrown when test response times out
- FlareConfig uses fluent builder pattern with `static` return types
- ConnectionError exception wraps all sender connection failures
- Code style: always use curly braces, happy path last, avoid else
